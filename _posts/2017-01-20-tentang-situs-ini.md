Sebuah situs lagi, tapi kali ini berbeda. [Situs ini](https://alexforsale.github.io/) berfungsi bukan sebagai ajang mencari ketenaran, karena, tenar itu apa sih? Bukan pula situs yang berfungsi mencuri content dari blog lain, terjemahkan ke bahasa Indonesia, lalu akui sebagai milik saya sendiri. Oke, lepas dari stigma blog Indonesia itu, fungsi saya membuat blog ini adalah:

* Belajar git
Blog ini di hosting oleh [github](https://github.com). Jadi tidak seperti blog - blog lainnya yang editornya lebih what you see is what you get, blog ini memaksa saya untuk menggunakan git dalam editingnya. Dan praktek terbukti adalah cara pembelajaran yang sangat efektif.
* Belajar Markdown.
Dengan mengisi post diblog ini, saya mau tidak mau harus menggunakan markdown, walaupun sebenarnya blog ini dapat langsung menggunakan HTML, sepertinya lebih seru jika belajar markdown.

Jadi apa yang akan menjadi isi dari blog ini? Jelasnya hanya akan berisi hal - hal yang menjadi minat saya pribadi. Dan tidak mesti berbau teknologi, karena siapa tau profesi menjadi _dukun_ masih terbuka...

*[HTML]: HyperText Markup Language
*[markdown]: is a lightweight markup language with plain text formatting syntax designed so that it can be converted to HTML and many other formats using a tool by the same name. Markdown is often used to format readme files, for writing messages in online discussion forums, and to create rich text using a plain text editor.
