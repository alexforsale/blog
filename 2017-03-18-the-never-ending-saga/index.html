What Dies Should Stay Dead

Posted on March 18, 2017

Akhir - akhir ini saya menemukan satu titik dimana kemalasan untuk menulis kembali datang, bukan berarti ini adalah hal yang baru, hanya saja rasa malas ini memandulkan banyak ide - ide bagus yang sebenarnya bisa dijadikan bahan tulisan. Dan the so-called-virus bernama kemalasan ini menyebar dengan skala yang menakutkan. Jadi untuk mengatasi-nya saya menggunakan cara old-school yang cukup efisien:’keep writing’.

So what have I been up to? jawabannya: “alot”, sebagai permulaan, ada satu hal yang sedang menarik minat saya dalam skala yang luar biasa: Emacs, tadinya saya hanya mencari editor alternatif dengan pilihan akhir antara emacs atau vim. Kenapa hanya dua itu pilihannya?

Tentu saja pilihan text editor tidak hanya emacs atau vim saja, tetapi akhirnya saya memilih emacs dengan beberapa alasan:

  • Emacs sangat fleksibel dan sangat mudah dikustomisasi. Emacs dapat dieksekusi langsung didalam terminal, atau menggunakan GUI tersendiri.
  • Sangat keyboard-oriented, ini dapat dimengerti mengingat emacs diciptakan sekitar tahun 1976, mouse belum trendy kala itu. Ini juga yang membuat proses editing teks menjadi lebih cepat begitu kita mulai terbiasa dengan perintah - perintah dan key-binding emacs.
  • Emacs memiliki lebih dari 1 clipboard, ini fitur yang sangat bagus, jadi ketika kita copy sesuatu kedalam clipboard, konten yang berada di_clipboard_ sebelumnya tidak hilang.
  • Konsep yang sama dalam membuka file, emacs memungkinkan kita untuk membuka beberapa file sekaligus.

Tentu tidak hanya itu fitur emacs yang tersedia, dan juga sebenarnya mungkin editor lain memiliki keunggulan yang sama atau lebih baik. Ditambah lagi, emacs dan vim bukanlah teks editor yang dapat digunakan langsung, butuh waktu untuk membiasakan dan mempelajarinya. Tetapi ini semua kembali ke preferensi masing - masing. Dan bagi saya dua teks editor ini (emacs dan vim), jauh lebih unggul dibandingkan yang lain.
